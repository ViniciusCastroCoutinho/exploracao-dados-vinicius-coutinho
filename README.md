# exploracao-dados-vinicius-coutinho
Análise do preço da gasolina no primeiro semestre de 2025, usando o dataset da ANP:  
https://www.gov.br/anp/pt-br/centrais-de-conteudo/dados-abertos/serie-historica-de-precos-de-combustiveis  
- Estísticas sobre o preço nacional e o preço do município de Manaus
- Ranking dos 10 estados com os maiores preços
- Ranking das 10 bandeiras com os maiores preços
- Ranking dos preços das bandeiras de Manaus
- Comparação do preço entre diferentes regiões
